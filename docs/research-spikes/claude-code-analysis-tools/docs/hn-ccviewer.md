# Show HN: Visualize full Claude Code CLI sessions in a rich UI

**Submitter:** rcanand2025
**URL:** https://rcanand.gumroad.com/l/ccviewer
**Score:** 1 point | 1 comment

## Content Summary

The creator describes this as a "scratch my own itch" project addressing a genuine pain point: Claude Code sessions are stored across multiple JSON files that are difficult to review later.

### Key Features
- Detailed visualization of Claude interactions including thinking processes, tool calls, code generation, and diffs
- Search, sort, and filter capabilities
- Markdown export functionality for sessions, repositories, or entire history
- Fully local, privacy-focused single-page HTML application

### Usage
Users download the HTML page and open it in Chromium-based browsers (Chrome, Edge). The tool reads sessions from the default location (~/.claude on Mac).

### Important Limitation
Requires Chromium-based browser — does not support Safari or Firefox.
