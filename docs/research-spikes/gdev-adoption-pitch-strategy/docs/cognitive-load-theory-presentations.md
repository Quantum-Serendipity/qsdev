# Cognitive Load Theory for Presentations

## Core Concept

"Cognitive load theory (CLT) implies that our brains can only retain so much information." Working memory has limits, and presenters must respect this threshold to engage audiences effectively.

## Three Types of Cognitive Load

**Intrinsic Load (Difficulty Level)**
The inherent complexity of content affects how audiences process information. Presenters should simplify material by breaking slides into manageable sections and highlighting key takeaways rather than overwhelming viewers with dense text.

**Extraneous Load (Presentation Method)**
How information is delivered significantly impacts comprehension. Visual representations substantially outperform text-heavy approaches — "when you choose to go visual with your information retention will increase by 42%." Converting descriptive text into visual formats reduces unnecessary cognitive strain.

**Germane Load (Building Blocks)**
Audiences learn better when presenters establish connections between concepts. Using visual storytelling and pairing relevant imagery with content helps viewers understand how ideas relate, creating mental frameworks for retention.

## Key Recommendations

1. Simplify content and remove unnecessary details
2. Prioritize visuals over text-based explanations
3. Connect concepts through coherent visual narratives
4. Respect audience mental capacity limits
5. Focus on key messages rather than comprehensive coverage

The article emphasizes that effective presenters demonstrate respect by acknowledging these cognitive constraints.
