# Keeping an Offline Copy of StackOverflow with Kiwix (2020)

## Setup Process
1. Download Kiwix Reader
2. Obtain ZIM packages via BitTorrent from official content directory
3. Open files in application

## Hardware Requirements
- Cheap USB mass storage devices
- ZIM files are "often several hundred GB in size"

## User Experience
- "Blazing fast load time" compared to web version
- Interface mirrors online version
- Good for use during transit or limited connectivity

## Filtering
No mention of any filtering or subsetting capabilities. Users download complete ZIM packages only.
