# Awesome Claude Code - Hooks Section

## Hooks-Related Tools & Libraries

### Britfix
- **URL**: https://github.com/Talieisin/britfix
- **Creator**: Talieisin
- **Description**: Automatically converts American English spellings to British English through a Claude Code hook, intelligently handling code files by converting only comments and docstrings while preserving identifiers and string literals.

### CC Notify
- **URL**: https://github.com/dazuiba/CCNotify
- **Creator**: dazuiba
- **Description**: Provides desktop notifications for Claude Code activities, alerting users when input is needed or tasks complete, with one-click integration back to VS Code and duration tracking.

### cchooks
- **URL**: https://github.com/GowayLee/cchooks
- **Creator**: GowayLee
- **Description**: A lightweight Python SDK offering a clean API and documentation to simplify hook development, abstracting away JSON configuration complexity.

### Claude Code Hook Comms (HCOM)
- **URL**: https://github.com/aannoo/claude-hook-comms
- **Creator**: aannoo
- **Description**: Enables real-time communication between Claude Code sub-agents using hooks, supporting @-mention targeting, live dashboard monitoring, with zero external dependencies.

### parry
- **URL**: https://github.com/vaporif/parry
- **Creator**: Dmytro Onypko
- **Description**: A prompt injection scanner for Claude Code hooks that scans tool inputs and outputs for injection attacks, secrets, and data exfiltration attempts (early development phase).
