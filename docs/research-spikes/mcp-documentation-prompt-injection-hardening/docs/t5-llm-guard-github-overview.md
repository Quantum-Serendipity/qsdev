# LLM Guard - GitHub Overview
- **Source**: https://github.com/protectai/llm-guard
- **Retrieved**: 2026-05-14

## Scanners Provided

LLM Guard offers two categories of scanners:

**Prompt Scanners** (15 types): Anonymize, BanCode, BanCompetitors, BanSubstrings, BanTopics, Code, Gibberish, InvisibleText, Language, PromptInjection, Regex, Secrets, Sentiment, TokenLimit, and Toxicity.

**Output Scanners** (20 types): Including BanCode, BanCompetitors, Bias, Code, Deanonymize, JSON, Language, MaliciousURLs, NoRefusal, and others for monitoring LLM responses.

## Prompt Injection Scanner

The web content lists "PromptInjection" as a supported scanner but provides no technical details about its mechanism, underlying models, or detection methodology on the main page.

## Deployment Options

LLM Guard supports multiple deployment approaches: Python library and API server. Installation uses standard package management: `pip install llm-guard`.

## System Requirements

Python version 3.9 or higher. 517 Commits at time of retrieval, suggesting active development.
