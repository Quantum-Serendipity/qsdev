# Atlassian — State of Developer Experience Report 2025

- **Source URL**: https://www.atlassian.com/teams/software-development/state-of-developer-experience-2025
- **Retrieved**: 2026-03-20
- **Note**: Landing page only; full report behind download gate. Key statistics from search results.
- **Sample**: 3,500 developers and managers (Wakefield Research)

## Key Statistics (from search result summaries)

- **69%** of developers report losing **8+ hours per week** to inefficiencies (from 2024 report)
- AI adoption is rising but organizational inefficiencies are increasing
- Biggest productivity drains: technical debt, insufficient documentation, complex build processes, lack of focus time
- Widening disconnect between leadership expectations and developers' actual experiences
- Only **38%** report significant productivity improvements from AI tools
