# Show HN: Subtle – Local, open-source analytics for Claude Code sessions

**Submitter:** itsderek23
**Repository:** https://github.com/itsderek23/subtle
**Points:** 2
**Posted:** ~January 12, 2026

## Key Features
- See overall Claude Code usage over time (broken down by AI vs tool time)
- Track AI-assisted Git commits over time
- Visualize individual sessions with execution traces
- Filter sessions by message content
- Access complete session transcripts

## Tool Details
- Free, open-source application
- Runs entirely locally on users' machines

## Developer's Reflection
The creator expressed uncertainty about optimal Claude Code usage patterns, asking the community: "fewer interventions? Smaller commits and PRs? Lower code churn within a session?" This suggests the tool aims to establish best practices for AI-assisted development workflows.
