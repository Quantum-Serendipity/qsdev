# Hetzner Storage Box Pricing & Specifications
- **Source**: https://www.hetzner.com/storage/storage-box/
- **Retrieved**: 2026-05-14

## Pricing Plans

| Plan | Storage | Protocols |
|------|---------|-----------|
| BX11 | 1 TB | FTP, FTPS, SFTP, SCP, Samba/CIFS, BorgBackup, Restic, Rclone, rsync via SSH, HTTPS, WebDAV |
| BX21 | 5 TB | Same |
| BX31 | 10 TB | Same |
| BX41 | 20 TB | Same |

BX31 is known to cost ~EUR 20.80/month for 10 TB (~EUR 2.08/TB/month).

## Key Details

- NFS is NOT supported (Samba/CIFS is the closest file share protocol)
- Unlimited traffic with free internal traffic
- 10 concurrent connections
- 100 sub-accounts
- Flexible upgrade/downgrade
- Storage locations: Germany or Finland
