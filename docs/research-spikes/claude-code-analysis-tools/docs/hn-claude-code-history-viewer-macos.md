# Show HN: Claude Code History Viewer for macOS

**Submitter:** jackleee
**Tool URL:** https://github.com/jhlee0409/claude-code-history-viewer/releases
**Posted:** ~July 2025

Desktop application built with Tauri + React + Rust. Reads conversation logs from `.claude` folder and presents them visually. Operates entirely locally for privacy. MIT license.

Creator noted: "I have a habit of reviewing my coding history when working with AI."
