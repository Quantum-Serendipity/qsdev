# [Remote] Help Shopify Rebuild Our World in Nix - NixOS Discourse

**Author:** burke (Burke Libbey)
**Date:** June 8, 2020
**Category:** Jobs

## Position Overview

Burke was recruiting for Shopify's Nix infrastructure team. He highlighted his background speaking at NixCon 2019 and presenting at Shopify ShipIt about the company's development tooling initiatives.

## Key Responsibilities & Goals

The role focused on developing "Runix" (a Nix-module-based system for declaring projects abstractly) and expanding Nix adoption across Shopify. Specific objectives included:
- Completing development environment rebuilds
- Implementing Nix in CI and Production systems
- Integrating Nix organization-wide

## Scale & Current Status (as of June 2020)

"We have thousands of R&D folks running Nix on their MacBook Pros already," indicating substantial existing adoption. The goal was making Nix the "common substrate" for the entire organization.

## Application Process

Interested candidates should email burke at shopify.com. The position was explicitly remote-eligible following Shopify's "digital by default" policy.

## Technical Details

Runix uses Nix modules to define standardized project types with inferred defaults, allowing developers to override specific configurations while treating divergences as opportunities for improvement.

Later replies questioned Runix's availability and existence, with burke clarifying it remained private during development, and he provided a GitHub gist demonstrating example project configurations.
