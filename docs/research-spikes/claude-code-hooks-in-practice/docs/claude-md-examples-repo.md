# ArthurClune/claude-md-examples Repository
- **Source**: https://github.com/ArthurClune/claude-md-examples
- **Retrieved**: 2026-03-27

## Overview

113 stars, 16 forks. Contains sample CLAUDE.md files designed for use with Claude Code.

## Files Included

1. **hugo-CLAUDE.md** — Configuration example for Hugo projects
2. **python-CLAUDE.md** — Configuration example for Python projects
3. **python2-CLAUDE.md** — Alternative Python configuration example
4. **terraform-CLAUDE.md** — Configuration example for Terraform projects

## Source Attribution

Examples were "edited/merged from other examples" including:
- modelcontextprotocol/python-sdk
- p33m5t3r/vibecoding (conway directory)
- saaspegasus/pegasus-docs

## Significance

One of the earliest collections of CLAUDE.md examples, showing the pattern of language/framework-specific configuration templates.
