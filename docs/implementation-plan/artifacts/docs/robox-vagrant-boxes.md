# Robox Generic Vagrant Boxes
> Source: https://github.com/lavabit/robox
> Retrieved: 2026-05-12

## Supported Distributions

### Linux
- Alpine Linux: 3.5-3.17
- Arch Linux
- AlmaLinux: 8, 9
- CentOS: 6, 7, 8, 8s, 9s
- Debian: 8-11
- Devuan: 1-4
- Fedora: 25-37
- Oracle Linux: 7, 8
- RHEL: 6, 7, 8
- Rocky Linux: 8, 9
- Ubuntu: 16.04-23.04
- openSUSE: 15, 42

### BSD
- FreeBSD: 11, 12, 13
- OpenBSD: 6, 7
- NetBSD: 8, 9
- DragonFly BSD: 5, 6
- HardenedBSD: 11, 12, 13

## Providers
VirtualBox, libvirt, VMware, Parallels, Hyper-V
Docker: only subset of configurations

## ARM64 Status
NO ARM64 variants. Documentation says "WE NEED A HARDWARE DONATION TO MAKE ARM64 IMAGES"

## Maintainer
Ladar Levison, Lavabit LLC

## Notable Gaps for gdev testing
- No Void Linux
- No Gentoo
- No NixOS
- No Manjaro/EndeavourOS/Pop!_OS/Linux Mint/Garuda
- Versions are older (Ubuntu stops at 23.04, Fedora at 37, Alpine at 3.17, Debian at 11)
- No ARM64 at all
