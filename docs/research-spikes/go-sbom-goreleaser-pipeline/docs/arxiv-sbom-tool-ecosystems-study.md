# The State of the SBOM Tool Ecosystems: A Comparative Analysis of SPDX and CycloneDX

## Citation
arXiv:2512.21781, December 2025

## Methodology
Quantitative comparative approach analyzing:
- **170 publicly advertised SBOM tools** across use cases
- **641 total tools** for ecosystem health metrics (171 CycloneDX, 470 SPDX)
- **36,990 issue reports** from open-source tools
- **Top 250 open-source projects** using each ecosystem

## Key Findings

**Ecosystem Size:**
- SPDX: 470 tools
- CycloneDX: 171 tools
- SPDX has roughly 2.75x more tools

**Ecosystem Characteristics:**
- CycloneDX demonstrates "higher developer engagement and certain health indicators"
- SPDX benefits from "a more mature ecosystem with broader tool availability and established industry adoption"

## Conclusions
The study identifies "complementary strengths" in both formats and highlights "opportunities for mutual enhancement" between SPDX and CycloneDX ecosystems, offering guidance for developers and practitioners selecting SBOM tools.
