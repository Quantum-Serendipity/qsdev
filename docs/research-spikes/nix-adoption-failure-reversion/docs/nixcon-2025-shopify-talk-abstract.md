# NixCon 2025 Talk: Nix-based Development Environments at Shopify (Reprise)

**Speaker:** Josh Heinrichs (working on developer tooling at Shopify from Saskatoon, Canada)

**Date & Time:** September 5, 2025, 16:10–16:30

**Location:** Lecture Hall (5.002)

**Duration:** 20 minutes

**Description:**
The presentation discusses Shopify's journey with Nix-based development environments. The company previously attempted Nix adoption in 2019 but the effort stalled. Recently, interest was rekindled through devenv. The talk covers the incremental migration strategy across their extensive project portfolio, transitioning from cloud to local development, multirepo to monorepo structures, and replacing Homebrew/Apt with Nix.

**Key Topics Covered:**
- Reasons the initial 2019 effort discontinued
- How devenv revitalized Nix interest
- Gradual migration approach for numerous projects
- Current implementation status and lessons learned
- Benefits Nix has delivered
- Challenges supporting a large, polyglot organization with developers across all stack layers
- Future direction
