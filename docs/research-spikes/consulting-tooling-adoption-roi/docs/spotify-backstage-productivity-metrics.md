## Key Metrics (from search results)

- Frequent Backstage users are **2.3x more active** in GitHub
- Frequent users **deploy twice as often**
- Changes take **17% less time** to go live
- Potential saving: equivalent of **3 full-time employees** in a team of 10 developers
- Backstage users' changes stay in production **3x longer** (suggesting higher code quality)
- Frequent users create **2x more code changes** than infrequent users

## Measurement Categories

Spotify measures Backstage success across:
- Developer productivity (work completed vs. effort/time invested)
- Developer efficiency (resources per unit of output)
- Developer effectiveness (code quality)
- Supplementary: time for new starters to merge 10th PR, plugin contributions, search success, context switching
