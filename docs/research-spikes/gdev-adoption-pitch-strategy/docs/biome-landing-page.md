# Biome Landing Page
- **Source URL**: https://biomejs.dev
- **Retrieved**: 2026-05-15

## Hero Section & Value Proposition
The landing page leads with "Biome, Toolchain of the web" and emphasizes its core benefit: "Format, lint, and more in a fraction of a second." The primary call-to-action buttons are "Get started" and "View on GitHub."

## Key Messaging

**Formatting capability:** Biome positions itself as "a fast formatter" for JavaScript, TypeScript, JSX, TSX, JSON, HTML, CSS and GraphQL with "97% compatibility with Prettier," claiming to save "CI and developer time."

**Linting features:** Described as "a performant linter" with "502 rules" sourced from ESLint, TypeScript ESLint, and others, emphasizing "detailed and contextualized diagnostics."

## Competitive Positioning
Performance metrics highlight significant speed advantages: "~35x Faster than Prettier when formatting 171,127 lines of code in 2,104 files with an Intel Core i7 1270P."

## Social Proof Elements
- **Enterprise adoption:** Logos from AWS, Google, Microsoft, Cloudflare, Discord, Vercel, and Node.js Foundation
- **Community:** Displays contributor avatars and maintains presence across Discord, GitHub, BlueSky, and Mastodon
- **Recognition:** Features "JS OS Award 2024 - Productivity booster"

## Above-the-Fold Structure
The page immediately presents the product name, visual tagline, core benefit statement, dual CTAs, code formatting example with output comparison, and performance metrics — all designed to establish credibility and demonstrate value before scrolling.
