# Datadog MCP Server Use Cases (Official)

## Official Status
Official from Datadog. Transitioned from Preview to GA in March 2026.

## Four Primary Use Cases

1. **Onboarding & Best Practices**: Identify monitors/dashboards developer uses, recommend proven patterns, generate IaC for monitoring setups
2. **Infrastructure Optimization**: Periodically identify unused services by filtering synthetic traffic from real activity, auto-create tickets for decommissioning
3. **Incident Root Cause Analysis**: Correlate alert timing with feature flag deployments to identify triggers, surface relevant metrics to on-call
4. **Cost Anomaly Detection**: Monitor cloud cost metrics, auto-alert service owners on spending spikes

## Supported Platforms
Claude Code, Cursor, Codex, Goose, GitHub Copilot, and other development environments.

## Practical Value
Reduced manual investigation, faster onboarding, lower cloud bills, quicker incident resolution through automated context gathering.
