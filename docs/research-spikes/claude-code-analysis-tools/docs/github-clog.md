# clog - Claude JSONL Log Viewer

Web-based application for visualizing and monitoring Claude Code conversation logs. No installation beyond opening an HTML file in a modern browser.

## Key Features

**Rich Log Display**
- Sessions organized with metadata visibility
- Messages show parent-child conversation relationships
- Token usage metrics include input/output counts and cache statistics
- Tool interactions display with visual indicators
- Supports text, tool_use, and tool_result message formats

**Real-Time Monitoring**
- Detects file changes automatically
- Refreshes content when new entries appear
- Optional auto-scroll to latest messages
- Visual status indicators for monitoring state

**Cross-Platform Support**
- Windows, macOS, Linux
- Browser compatibility: Chrome, Edge, Firefox, and Safari

## Technical Foundation
Built with vanilla JavaScript (no dependencies), HTML5, and Tailwind CSS. Leverages File System Access API for advanced features on Chrome/Edge.
