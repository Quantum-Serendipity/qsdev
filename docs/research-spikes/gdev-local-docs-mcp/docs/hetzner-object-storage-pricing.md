# Hetzner Object Storage Pricing
- **Source**: https://www.hetzner.com/storage/object-storage/
- **Retrieved**: 2026-05-14

## Base Included Quota (Per Month)

- 1 TB storage (up to 744 TB-hours)
- 1 TB egress traffic

Reported at approximately $5.20/TB/month with full versioning, object lock, $0.01/GB egress (with 1 TB free internal transfer).

## Free Traffic

- Ingress (uploads): free
- Internal traffic within eu-central network zone: free
- S3 API calls (PUT, GET, DELETE): free

## Storage Specifications

- Up to 100 buckets, 100 TB each
- Minimum billable object size: 64 KB
- Maximum objects per bucket: 50 million

## Supported Regions

- Falkenstein, Germany (FSN1)
- Helsinki, Finland (HEL1)
- Nuremberg, Germany (NBG1)

## Key Features

S3-compatible, GDPR-compliant, object locking, encryption, versioning, pre-signed URLs.
