# fzf - command-line fuzzy finder

fzf is described as "a general-purpose command-line fuzzy finder" that functions as "an interactive filter program for any kind of list; files, command history, processes, hostnames, bookmarks, git commits, etc."

## Key Highlights (4 bullet points at top)

- **Portable** -- distributed as a single executable for straightforward setup
- **Fast** -- engineered to handle millions of items with instant processing
- **Versatile** -- entirely configurable through event-action binding mechanisms
- **All-inclusive** -- ships with integrations for Bash, Zsh, Fish, Vim, and Neovim

## Installation Options

Multiple platforms including Homebrew, Linux package managers (APT, DNF, Pacman, etc.), Windows tools (Chocolatey, Scoop, Winget), and direct Git cloning.

## Core Features

Fuzzy matching algorithm enabling users to enter patterns with omitted characters. Multiple display modes including standard fullscreen, `--height` mode, and `--popup` mode for tmux/Zellij.

## Key Bindings (Shell Integration)

- `CTRL-T` pastes selected files and directories
- `CTRL-R` retrieves command history selections
- `ALT-C` changes into selected directories

## Advanced Capabilities

Preview window functionality with ANSI color support, image display protocols (Kitty, iTerm2, Sixel), and dynamic list reloading through binding mechanisms.
