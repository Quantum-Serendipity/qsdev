# GitHub Enterprise — Developer Onboarding & Productivity ROI

- **Source URL**: https://github.blog/enterprise-software/collaboration/increase-developer-productivity-save-time-on-developer-onboarding-and-drive-roi-in-2023/
- **Retrieved**: 2026-03-20
- **Study scope**: Composite organization of 120,000 employees, 7,000 developers, 20,000 repos

## Developer Onboarding

- **80% reduction** in time spent training new developers
- Onboarding reduced from **10 days to minutes** (TELUS case study)
- New developers can "make an impact from day one"

## Productivity Gains

- **433% ROI** over three years using GitHub Enterprise Cloud + Advanced Security
- **22% developer productivity increase** after three years
- **2 hours saved per developer per week** (TELUS example)
- **75% tool maintenance productivity gains** by phasing out legacy systems
- **1 minute saved per GitHub Actions execution** (TELUS)

## Cost Savings

- **$2.7M** saved in IT auditor costs over three years
- **$5.2M** saved in code remediation time
- **$811,900** total cost of ownership savings over three years (retiring legacy tools)
- **15% efficiency improvement** for DevOps/DevSecOps teams
