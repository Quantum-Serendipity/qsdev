# Show HN: Kintsugi – A desktop app for reviewing Claude Code sessions

**Submitter:** vikvorona (engineer at Sonar)
**Tool URL:** https://events.sonarsource.com/kintsugi/

## Overview

Complementary IDE-like application for Claude Code emphasizing code quality and security review. "We want to ship fast and Claude Code is really-really good at this. So we decided to create a tool that would allow developers to ship fast, but have control over the produced code."

## Key Features

- Orchestrate multiple AI agents with session status tracking
- Review AI-generated code similar to pull requests with comments and pushback
- Formatted plan review with inline comments
- Integrated SonarQube analysis for code quality

## Notes

- Built almost entirely by Claude Code
- Currently macOS-only
- Requires Java 17+ (friction point noted in comments)
- By Sonar (makers of SonarQube)
