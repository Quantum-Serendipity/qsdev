# Mantra - Time Travel for AI-Assisted Programming

Desktop application that transforms AI coding sessions into navigable, replayable timelines. Scrub through sessions like video playback, observing code evolution across conversation turns. Operates locally without accounts or external data.

## Core Features

- **Time Travel**: Replay sessions as visual timelines with forward/backward scrubbing
- **Project Organization**: Consolidate sessions from multiple AI tools
- **MCP Hub**: Manage MCP servers, tools, and permissions across agents
- **Skills Hub**: Create/distribute reusable slash commands across tools
- **Search & Filtering**: Locate specific turns where code changes were introduced
- **Content Protection**: Redact sensitive information before sharing
- **Flexible Viewing**: Toggle between conversation, code-change, or file-tree modes
- **Privacy-First**: All data local, no cloud, no telemetry

## Supported Tools

- Claude Code, Codex, Gemini CLI (since v1.0.0)
- Cursor (since v0.35.0, requires 0.40.0+)
- Antigravity (forthcoming)

## System Requirements

- macOS 12.0+, Windows 10 (build 1903+), Linux (Ubuntu 20.04+/Fedora 35+/Debian 11+)
- 4GB RAM, 500MB disk
