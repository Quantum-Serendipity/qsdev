# GCP Cloud Storage Pricing 2026
- **Source**: https://leanopstech.com/blog/google-cloud-storage-pricing-2026/
- **Retrieved**: 2026-05-14

## Storage Rates (Single Region, us-central1)

| Storage Class | Cost per GB/month | Cost per TB/month |
|---|---|---|
| Standard | $0.020 | $20.00 |
| Nearline | $0.010 | $10.00 |
| Coldline | $0.004 | $4.00 |
| Archive | $0.0012 | $1.20 |

Standard is the only class where single-region pricing differs from multi-region at 23% savings.

## Operation Costs (Per 10,000 Operations)

| Storage Class | Class A (writes/lists) | Class B (reads/gets) |
|---|---|---|
| Standard | $0.05 | $0.004 |
| Nearline | $0.10 | $0.01 |
| Coldline | $0.10 | $0.05 |
| Archive | $0.50 | $0.50 |

## Egress Pricing

Internet egress from us-central1 ranges from $0.08-$0.12 per GB depending on volume. Highest rate ($0.12/GB) applies to first terabyte monthly.
