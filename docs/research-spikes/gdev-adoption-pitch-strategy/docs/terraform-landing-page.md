# Terraform Landing Page
- **Source URL**: https://developer.hashicorp.com/terraform
- **Retrieved**: 2026-05-15

## Hero Section
**Main Headline:** "Automate Infrastructure on Any Cloud"

**Opening Value Proposition:** "Terraform is an infrastructure as code tool that lets you build, change, and version infrastructure safely and efficiently."

## Above the Fold Elements

The page features a prominent hero image alongside three primary call-to-action buttons:
- Install
- Tutorials
- Documentation

## Core Value Proposition
The tool manages "low-level components like compute instances, storage, and networking; and high-level components like DNS entries and SaaS features."

## Primary Call-to-Action
"Try HCP Terraform for free" - directing users to the hosted cloud platform offering.

## Visual & Structural Elements
- Hero banner with dark/light mode imagery
- Three-column call-to-action layout immediately below headline
- "Get Started" section with provider-specific tutorials (AWS, Azure, GCP, Oracle, Docker)
- Sandbox environment offering for hands-on experimentation

## Social Proof & Support
The page lacks traditional testimonials or numerical metrics. Instead, it emphasizes:
- Multi-cloud provider support (6 major cloud platforms featured)
- Free trial access to HCP Terraform
- Certification programs and learning paths
- Featured documentation and use cases

## Competitive Positioning
Emphasis on "any cloud" flexibility and enterprise-grade collaboration features through HCP Terraform.
