# Stack Exchange Data Dump - Archive.org File Listing

## Overall Structure
The dump consists of 371 separate files totaling 92.3GB. Each Stack Exchange site is archived individually as a compressed 7z file using bzip2 compression.

## Stack Overflow Files (Largest)
- stackoverflow.com-Posts.7z: 21.4GB
- stackoverflow.com-PostHistory.7z: 35.2GB
- stackoverflow.com-Comments.7z: 6.5GB
- stackoverflow.com-Votes.7z: 2.1GB
- stackoverflow.com-Users.7z: 944.3MB
- stackoverflow.com-Badges.7z: 514.7MB
- stackoverflow.com-PostLinks.7z: 144.4MB
- stackoverflow.com-Tags.7z: 1.1MB

## Archive Contents per Site
Each site archive typically includes Posts, Users, Votes, Comments, Badges, Tags, PostHistory, and PostLinks in XML format.

## Total Size
Complete dataset is approximately 92.3 gigabytes across all Stack Exchange network sites.
