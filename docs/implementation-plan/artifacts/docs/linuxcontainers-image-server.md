## VM-Only

FreeBSD

## Key Notes

- Both container and VM images available for major distros
- ARM64 support across all major distros
- NixOS images available (container + VM)
- Void Linux is container-only
- Arch Linux is container-only (no VM image)
- Gentoo has both container and VM
