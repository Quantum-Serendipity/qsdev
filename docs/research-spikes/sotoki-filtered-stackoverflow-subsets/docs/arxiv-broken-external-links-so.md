# Broken External Links on Stack Overflow (2020)

## Key Statistics

- **11,926,354 links** referencing third-party websites on Stack Overflow
- **14.2%** of these external links are broken
- 65% of broken links in questions served as code examples
- 70% of broken links in answers provided supporting information
- Only 1.67% of posts with broken links had the issue highlighted in comments
- Only 5.8% of posts with broken links subsequently removed them
- Web technologies (JavaScript, HTML, CSS, jQuery) had more broken links

## Notes

- This study focuses on EXTERNAL links only (third-party websites)
- Does not provide internal (SO-to-SO) link statistics
- Does not break down internal vs external link counts
