**Submitter:** yurukusa
**Tool URL:** https://yurukusa.github.io/cc-toolkit/
**Posted:** ~25 days ago | 1 point

## Overview

41 zero-dependency tools built over 60 days to analyze Claude Code usage patterns. Browser-based, no installation required, MIT-licensed.

## Featured Tools

- **cc-wrapped**: Visual yearly summary of Claude Code activity
- **cc-session-stats**: Session duration tracking with break reminders
- **cc-agent-load**: Comparative time split between user and AI subagents
- **cc-ghost-log**: Identifies days with code commits but no recorded sessions
- **cc-impact**: Aggregates commits, lines added, and file modifications
- **cc-peak**: Hourly focus patterns and optimal work windows
- **cc-burnout**: Risk assessment showing sustainability concerns
- **cc-predict**: Projects month-end statistics based on recent trends
