# Slack MCP Server Guide (Official)

## Official Status
Official Slack product.

## Available Tools & Actions

Four main capability categories:

1. **Search**: Search for messages, files, members, and channels
2. **Message Management**: Read and retrieve channel history and send messages to any type of conversation in Slack
3. **Canvas Operations**: Create and read Slack canvases
4. **Member Access**: Access and read profile information including custom profile fields and statuses

## Partner Integrations
12 supported partners: Claude, Credal AI, Cursor, Dropbox, Guru, Jasper, Notion, OpenAI, Perplexity, Thoughtspot, Wordsmith, and Workleap. Additional partners coming soon.

## Security & Permissions
Provides "secure access" via partner setup process. Specific permission model details in developer documentation.
