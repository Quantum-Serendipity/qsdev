# PyPI 2025 Year in Review: Key Statistics and Security Enhancements

## Scale and Growth Metrics
PyPI demonstrated substantial growth in 2025:
- **3.9 million** new files published
- **130,000** new projects created
- **1.92 exabytes** of data transferred
- **2.56 trillion** total requests served
- **81,000** requests per second on average

## Security Implementations

**Two-Factor Authentication Enhancements**
PyPI rolled out "email verification for TOTP-based logins," adding confirmation requirements for phishable authentication methods. Results included over 52% of active users enabling non-phishable 2FA and more than 45,000 verified logins.

**Trusted Publishing Expansion**
The platform extended support to GitLab Self-Managed instances and custom OIDC issuers for organizations. Adoption figures showed:
- Over 50,000 projects utilizing trusted publishing
- More than 20% of file uploads conducted via trusted publishers

**Attestations**
"17% of all uploads to PyPI in the last year included an attestation," representing verifiable claims about software provenance.

**Proactive Protections**
Implemented defenses included phishing detection, ZIP file security hardening, typosquatting flagging, domain resurrection prevention, and spam restrictions targeting specific email domains.

## Incident Response and Safety Operations

**Malware Management**
The platform processed over 2,000 malware reports with response speeds of 66% handled within 4 hours and 92% within 24 hours.

**Support Operations**
- 2,221 account recovery requests resolved
- Over 500 project name retention requests processed

## Organizations Adoption
- 7,742 organizations created
- 9,059 projects under organizational management
