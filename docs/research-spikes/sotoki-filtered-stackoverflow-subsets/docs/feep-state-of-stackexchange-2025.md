# The Current State of StackOverflow's Data Dumps (Feb 2025)

## Access & Distribution Changes (2024)
- Stack Exchange shifted from Internet Archive uploads to self-hosted distribution
- Users must log into each of 368 separate sites and manually download dumps
- Must agree to terms before downloading

## Licensing & Restrictions
- Agreement: "this file is being provided to me for my own use and for projects that do not include training a large language model (LLM)"
- Stack Overflow reserves right to deny future access if files distributed for LLM training
- They lack technical enforcement mechanisms

## Technical Details
- Dumps are NOT watermarked
- Official checksums provided with quarterly releases
- Recent dumps reported to contain missing posts from deleted users (data completeness concern)

## Community Mirrors
- Volunteers continue uploading to Internet Archive despite company requests not to
- CC BY-SA license permits redistribution
- Current dumps available at stackexchange_20241231 and similar collections
- Independent resource at communitydatadump.com
