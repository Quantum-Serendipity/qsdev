# Platform Engineering Case Study — Onboarding 2 Weeks to 2 Hours

- **Source URL**: https://platformengineering.com/features/how-we-reduced-onboarding-time-from-2-weeks-to-2-hours-a-platform-engineering-case-study/
- **Retrieved**: 2026-03-20

## Before & After

- **Before**: 2 weeks (14 days) per developer
- **After**: 2 hours per developer
- **Scale**: 50 new hires = 100 weeks of cumulative onboarding time reduced

## Root Cause Analysis

- **60%** of onboarding time spent waiting for approvals
- **30%** of setup steps were identical for all developers
- **25%** of issues related to environment inconsistencies
- **15%** of time spent fixing configuration errors

## Implementation

- 8-week implementation using Terraform and Docker
- Self-service portal for guided onboarding
- Automated role-based access control
- Infrastructure as Code eliminated environment inconsistencies
