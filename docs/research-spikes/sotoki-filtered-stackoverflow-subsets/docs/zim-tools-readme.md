# ZIM Tools - README

## Primary Tools

**zimcheck** — Verifies that a given ZIM file is not corrupted. Provides many features to secure that ZIM entries are proper and properly linked.

**zimdump** — Inspects or dumps (part of) a ZIM file, allowing users to examine ZIM file contents selectively.

**zimsplit** — Splits smartly a ZIM file in smaller chunks, using the official ZIM file format specifications for archive segmentation.

**zimwriterfs** — Creates ZIM files from a locally-stored directory containing self-sufficient HTML content (with pictures, javascript and stylesheets). It packages local directory contents into compressed ZIM archives suitable for offline reading applications.

## Additional Tools

The documentation mentions that "a few other tools are provided as well but are of less value for most of the usages," though specific names and descriptions for these secondary tools are not included in the provided content.

## Key Capabilities

The toolkit emphasizes functionality for verification (zimcheck), inspection (zimdump), file manipulation (zimsplit), and content packaging (zimwriterfs). These tools support the openZIM project's goal of creating and maintaining offline content archives.
