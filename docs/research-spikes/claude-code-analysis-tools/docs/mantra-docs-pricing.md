# Mantra Pricing — Documentation

## Core Offering
Core features $0 permanently free, no registration required; paid options are optional add-ons.

## Free Tier Features
All local functionality remains perpetually free with no restrictions:
- Unlimited sessions and projects
- Full-text search across sessions
- Git integration time travel
- Local redaction engine
- BYOK (Bring Your Own Key) support
- macOS, Windows, Linux compatibility

## Optional Paid Services

**Sync (Multi-device synchronization):**
- $4/month (annual) or $5/month (monthly)
- End-to-end encryption
- Cross-device session sync
- Cloud prompt and skills management
- 1-year version history

**Publish (Session sharing):**
- $8/month per site (annual) or $10/month per site (monthly)
- Web replay link generation
- Custom domains and branding
- Password protection
- Built-in search and themes

## Support Tiers

**Pioneer (one-time):**
- $49 (limited to 1,000 seats)
- Early feature access, founder badge, Discord privileges, 10GB encrypted backup, first-year Sync included, lifetime 50% Sync renewal discount

**Commercial (annual):**
- $50/person/year
- Priority email support, invoicing

## Business Model Notes
No investors, no ads, no tracking. Commercial licensing encouraged but not mandatory for organizational use.
