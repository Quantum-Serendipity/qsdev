# Ref.Tools MCP Server Summary

## Tools Provided
The Ref MCP Server offers two primary tools:

1. **`ref_search_documentation`** - Enables querying across indexed documentation resources, with support for programming language, framework, and library names

2. **`ref_read_url`** - Retrieves full content from documentation URLs, whether from search results or external web sources

## Documentation Access
"Fetch the complete documentation index at: https://docs.ref.tools/llms.txt" to discover all available pages. The system uses indexed documentation resources that can be searched and retrieved.
