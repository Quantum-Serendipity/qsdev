# Book Review: Presentation Patterns by Neal Ford, McCullough, Schutta

## Book Details
- **Title:** Presentation Patterns: Techniques for Crafting Better Presentations
- **Authors:** Neal Ford, Matthew McCullough, Nathaniel Schutta
- **Rating:** 82% of Amazon reviewers gave 5 stars

## Content Organization
The book contains 60+ patterns and 25+ antipatterns organized into three sections:

**Part I -- Prepare (25%):** Presentation planning and creativity
**Part II -- Build (50%):** Slide construction and temporal elements
**Part III -- Deliver (25%):** Stage preparation and performance

## Key Patterns Mentioned

- **Expansion Joints:** Building flexibility into presentations to accommodate time constraints
- **Unifying Visual Theme:** Using repeating visual elements like color schemes or imagery
- **Weatherman:** Delivering slides with gestures while facing the audience (avoiding reading from slides)
- **Live Demo:** Demonstration of working software as a presentation technique

## Key Antipatterns Mentioned

- **Dead Demo:** Demonstrating the tool consumes presentation time that could be packed with more information; even outstanding technical demonstrators cover at most only 60% of the volume of material that could be done as a presentation
- **Lipstick on a Pig:** Using flashy demo to cover up lack of substance
- **Floodmarks:** Cluttering slide headers/footers with logos and watermarks
- **Going Meta:** Discussing the presentation itself within the presentation
- **Ant Fonts:** Unreadably small text from overcrowding slides

## Reviewer's Assessment
Recommended particularly for those with technical backgrounds. "Aims to define a new vocabulary for presenters."

**Strengths:** Solid advice throughout; strong emphasis on slide transitions and temporal design
**Weaknesses:** Navigation challenges; line art symbols lack clarity; inconsistent typography/color usage
