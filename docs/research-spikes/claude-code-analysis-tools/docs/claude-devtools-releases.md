# Claude DevTools — Release History

## Releases (Oldest to Newest)

1. **v0.4.0** — February 14, 2025
   - Added Intel macOS support, Linux support, and WSL support for discovering Claude root path candidates

2. **v0.4.1** — February 16, 2025
   - Introduced API endpoints for hiding and unhiding individual and bulk sessions in configuration management

3. **v0.4.2** — February 19, 2025
   - Docker support, session management, context insights, and subagent display improvements

4. **v0.4.3** — February 19, 2025
   - Fixed Linux sandbox permissions issues

5. **v0.4.4** — February 20, 2025
   - Enhanced MCP tool input/output rendering, improved notification handling, added session context copy functionality

6. **v0.4.5** — February 21, 2025
   - Disabled default notification triggers, added global session search, introduced session export (Markdown/JSON/Plain Text), added subagent color badges

7. **v0.4.6** — February 24, 2025
   - Fixed sidebar auto-expand, resolved performance regressions, added auto-expand setting for AI responses, custom title bar on Linux

8. **v0.4.7** — February 26, 2025
   - Reverted several previous PRs and clarified project scope

9. **v0.4.8** — March 9, 2025
   - Fixed JSONL deduplication, added syntax highlighting for R/Ruby/PHP/SQL, optimized search performance

10. **v0.4.9** — March 23, 2025
    - Optimized message categorization, cached compiled regexes, added collapsible output sections, fixed Windows/WSL path handling

## Observations
- First public release: February 14, 2025 (v0.4.0 — suggesting prior private development)
- 10 releases over ~5 weeks (Feb 14 – Mar 23, 2025)
- Rapid iteration: multiple releases per week in first 2 weeks
- Pace slowed: 2-week gap before v0.4.8, then 2-week gap before v0.4.9
- All stable releases (no pre-releases)
- v0.4.7 included reverts, suggesting some feature scope management challenges
