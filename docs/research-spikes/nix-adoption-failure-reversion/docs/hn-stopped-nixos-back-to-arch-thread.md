# HN Thread: I stopped using NixOS and went back to Arch Linux
- **Source URL**: https://news.ycombinator.com/item?id=47339204
- **Retrieved**: 2026-03-20
- **Type**: Hacker News discussion thread

## Main Account

**User: birdculture** (Original post author / Sleeyax blog)
- **Switch:** NixOS -> Arch Linux
- **Duration:** ~1 year
- **Reasons:** Frequent build failures after updates, long compilation times (4-5+ hours for maintenance updates), large storage requirements from the Nix store, constant rebuild-fix cycles during configuration

## Secondary Perspectives

The thread contained mostly **defensive responses from NixOS users** rather than additional switchers:

- **craftkiller**: Long-time NixOS user highlighting advantages over Arch's rollback capabilities
- **nbf_1995**: Current NixOS user addressing criticisms
- **joshmoody24**: Enthusiast praising stability

No other commenters explicitly stated they abandoned NixOS. Several discussed hypothetical concerns (link rot reproducibility, configuration complexity) without confirming personal abandonment.
