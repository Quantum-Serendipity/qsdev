# Archive Team - Stack Exchange

## Recent Access Changes (2024)
On July 12, 2024:
- Public accessibility on archive.org was discontinued
- Dumps now require user login and download from individual account settings
- Users must accept new terms restricting use: "this file is being provided to me for my own use and for projects that do not include training a large language model (LLM)"
- Violation could result in losing future download access

## Dump Availability
- Published every few months at archive.org/details/stackexchange
- ZIM format versions available through Kiwix and Internet Archive
- stackprinter.com archives deleted questions
