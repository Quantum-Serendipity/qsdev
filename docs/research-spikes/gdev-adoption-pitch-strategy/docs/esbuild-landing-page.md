## Hero Section
**Main Heading:** "esbuild" with tagline "An extremely fast bundler for the web"

**First Sentence:** "Our current build tools for the web are 10-100x slower than they could be."

## Value Proposition
The primary pitch emphasizes speed as the core differentiator. Key features listed include:
- "Extreme speed without needing a cache"
- Built-in support for JavaScript, CSS, TypeScript, and JSX
- "Straightforward API for CLI, JS, and Go"
- ESM and CommonJS module bundling
- CSS modules support
- Tree shaking, minification, and source maps
- Local server, watch mode, and plugin system

## Social Proof & Competitive Positioning
A production bundle benchmark comparing esbuild (0.39s) against major competitors demonstrates dramatic speed advantages over Parcel 2 (14.91s), Rollup 4 (34.10s), and Webpack 5 (41.21s). The benchmark uses "10 copies of the three.js library" with "default settings, including minification and source maps."
