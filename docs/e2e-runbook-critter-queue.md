### PreToolUse Hook — Package Guard (Validation Tests)

Package install commands are already in the `ask` list by default (see A4-1 through A4-4 above). The PreToolUse package-guard hook intercepts them automatically. These tests exercise the hook's validation logic in more depth: OSV.dev CVE checks, age-gating, safety flag injection, and the denylist/allowlist env vars.

**A4-18b** — Safe package install (Claude prompt):
> Install the `helmet` package for Express security headers: `pnpm add helmet`

1. Detects `pnpm add helmet` as a package install command
2. Queries OSV.dev for known vulnerabilities in `helmet`
3. Checks npm registry — `helmet` is years old, well past the 3-day age gate
4. Returns `permissionDecision: "allow"` with safety context
5. Claude gets an "ask" prompt (from the ask rule) — you approve it
