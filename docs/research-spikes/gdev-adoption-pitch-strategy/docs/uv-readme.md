# uv README (GitHub)
- **Source URL**: https://github.com/astral-sh/uv
- **Retrieved**: 2026-05-15

**First Sentence:**
"An extremely fast Python package and project manager, written in Rust."

**Value Proposition:**
A unified tool replacing pip, pip-tools, pipx, poetry, pyenv, twine, virtualenv and more, delivering "10-100x faster" performance than pip.

**Installation Command:**
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

**Key Features:**
- Single tool consolidating multiple Python package management functions
- Comprehensive project management with universal lockfile
- Script execution with inline dependency metadata
- Python version installation and management
- Tool execution for Python packages
- Pip-compatible interface
- Cargo-style workspaces for scalable projects
- Global cache for disk efficiency
- Cross-platform support (macOS, Linux, Windows)

**Social Proof:**
- 85,000+ GitHub stars
- 3,100+ forks
- Used by 15,900+ projects

**Competitive Positioning:**
Backed by Astral, creators of Ruff and ty. Installable without requiring Rust or Python prerequisites.
