# Rust ZIM Library (dignifiedquire/zim)
> Source: https://github.com/dignifiedquire/zim
> Retrieved: 2026-05-14

## Overview
- **Repository:** dignifiedquire/zim (forked from eminence/zim)
- **Description:** "A rust library and cli tool to read and extract zim files."
- **Language:** 100% Rust (pure Rust, no C dependencies)
- **Stars:** 32
- **Forks:** 7
- **License:** Dual Apache 2.0 / MIT

## Features
- ZIM file reading and extraction
- IPFS integration for distributed content storage
- CLI tools: extract-zim and ipfs-link
- Link skipping during extraction (--skip-link flag)

## Build
```bash
cargo build --release
```

## Limitations
- Full-text search capabilities: not mentioned/likely not supported
- No Xapian integration (pure Rust)
- README details sparse in fetched content
