# Sotoki offliner-definition.json
- **Source URL**: https://raw.githubusercontent.com/openzim/sotoki/main/offliner-definition.json
- **Retrieved**: 2026-05-14

---

Defines the zimfarm offliner parameters. Confirms the complete set of supported flags:
domain, name, title, description, favicon, creator, publisher, tags (ZIM metadata),
without_images, without_user_profiles, without_external_links, without_unanswered,
without_users_links, without_names, censor_words_list, output, threads, tmp_dir,
zim_file, optimization_cache, mirror, stats_filename, redis_url, defrag_redis,
debug, keep_redis.

**No tag-based content filtering parameter exists in the offliner definition.**

Notable: mirror has a default value of https://s3.us-west-1.wasabisys.com/org-kiwix-stackexchange
