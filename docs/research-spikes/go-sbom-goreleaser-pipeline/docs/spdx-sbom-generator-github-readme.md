# SPDX SBOM Generator: Project Status Summary

## Repository Metrics
- **GitHub Stars**: 425
- **Forks**: 112
- **Status**: Archived (January 13, 2025)

## Deprecation Notice
The project is no longer maintained. The repository explicitly states: "Due to the un-availability of maintainers for the project, and the wide availability and support for other SBOM tools, this project is deprecated."

## Technical Details
- **Primary Language**: Go (97.5%)
- **Last Release**: v0.0.15 (July 12, 2022)
- **Supported Package Managers**: 11 ecosystems including GoMod, Cargo, Maven, NPM, Yarn, PIP, and others

## Recommended Alternatives
The SPDX community suggests using these actively maintained tools instead:
- **Trivy** - for SBOM generation
- **Parlay** - for SBOM enrichment with third-party data
- **Syft** - for archives and container SBOMs

## Historical Context
The tool was designed to generate SPDX v2.2 format software bills of materials across multiple package management ecosystems, supporting CI/CD integration through command-line tooling written in Go.
