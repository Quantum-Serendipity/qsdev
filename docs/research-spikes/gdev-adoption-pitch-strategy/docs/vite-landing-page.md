## Value Proposition
The page emphasizes four core features:
- Instant server start with native ESM and fast dependency pre-bundling
- "Lightning Fast HMR" for instant change reflection during development
- Rich built-in features (TypeScript, JSX, CSS, Workers, WebAssembly)
- Optimized build output via tree-shaking and minification
