# An Exploratory Study on the Repeatedly Shared External Links on Stack Overflow (2021)

## Key Statistics

- **11,926,354** external links reused on Stack Overflow
- **82.5%** of link sharing activities share external resources
- **57.0%** of external link occurrences are repeated external links (same URL shared multiple times)

## Derived Implication

If 82.5% of link sharing activities are external, then ~17.5% are internal (SO-to-SO).
This implies approximately 2.5M internal link instances in the dataset studied.

However, this ratio may not apply directly to PostLinks.xml, which tracks a different 
kind of relationship (sidebar links created by the system from inline references, plus
explicit duplicate markings) vs inline body text links.
