# Mantra - AI Coding Sessions Time Machine

## Overview

Mantra is a free, local-first desktop application that replays AI coding sessions from Claude Code, Cursor, and Gemini CLI. Enables developers to review AI-assisted coding work with timeline-based navigation.

## Core Features

- **Session Replay & Time Travel**: Click any message to instantly jump to the Git snapshot from that moment to review code states
- **MCP Hub**: Unified control plane for Model Context Protocol services across multiple AI tools
- **Skills Hub**: Available alongside other management features
- **Sensitive Data Detection**: Identifies API keys, database passwords, access tokens, and private keys
- **Full-Text Search**: Query across all imported sessions

## Supported Platforms & Tools

Runs on macOS (Intel/Apple Silicon), Windows 10/11, and Linux (Ubuntu 20.04+, Fedora 36+). Currently supports Claude Code, Cursor, and Gemini CLI, with OpenAI Codex and Antigravity support coming soon.

## Technical Details

- **Version**: 0.11.0
- **Built with**: Rust and React
- **Cost**: Free forever for core features; optional paid cloud add-ons for Sync and Publish
- **Privacy**: All core features operate 100% offline with end-to-end encryption for optional cloud features

## Key Use Case

Debugging AI-introduced bugs: when something breaks, scrub backward to find the exact point where the AI misunderstood your intent. Records structured data -- prompts, responses, tool calls, file changes -- and lets you move through them on a timeline (TimberLine controller).

## Data Privacy

"Your data never leaves your device" for offline functionality. Cloud features use encryption developers cannot decrypt.
