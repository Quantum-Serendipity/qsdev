# DX Newsletter — Developer Ideal vs Actual Workdays

- **Source URL**: https://newsletter.getdx.com/p/developer-ideal-and-actual-workdays
- **Retrieved**: 2026-03-20
- **Note**: Summary of Microsoft Time Warp study findings

## Key Data Points

- Sample: 484 software developers at Microsoft
- Coding: actual 11%, ideal 20% (gap of 9 percentage points)
- Architecting & designing: actual 6%, ideal 15% (gap of 9 percentage points)
- Communication & meetings: actual 12%, ideal much lower (largest negative gap)

## Automation Priorities

Top activities developers want automated:
1. Documentation
2. **Environment setup/maintenance** (#2 priority)
3. Testing
4. Task tracking/backlog management
5. Security and compliance

## Negative Impact Activities

Dev environment setup/maintenance has **statistically significant negative impact** on both productivity and satisfaction.
