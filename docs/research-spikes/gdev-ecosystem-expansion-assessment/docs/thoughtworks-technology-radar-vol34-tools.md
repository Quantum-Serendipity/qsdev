# ThoughtWorks Technology Radar Vol 34 - Tools

- **Source URL**: https://www.thoughtworks.com/en-us/radar/tools
- **Retrieved**: 2026-05-14

## Adopt Ring (Recommended for Industry Adoption)

1. **Axe-core** - Open-source accessibility testing tool for WCAG compliance checking
2. **Claude Code** - Agentic AI coding tool for multi-step workflow planning and execution
3. **Cursor** - Comprehensive agentic IDE environment with plan mode and subagent support
4. **Kafbat UI** - Free web UI for Apache Kafka cluster monitoring and message inspection
5. **mise** - High-performance Rust-based tool consolidating version management, environment variables, and task execution

## Trial Ring (Worth Exploring)

1. **cargo-mutants** - Mutation testing for Rust with zero-configuration approach
2. **Claude Code plugin marketplace** - Git-based distribution for AI commands and skills
3. **Dev Containers** - Standardized containerized development environments with sandboxing capabilities for coding agents
4. **Figma Make** - AI-powered UI prototyping using design system components
5. **OpenAI Codex** - Standalone agentic coding tool for autonomous task delegation
6. **Typst** - Modern markup-based typesetting system for programmatic document generation

## Assess Ring (Worth Understanding)

Agent Scan, Beads, Bloom, CDK Terrain, CodeScene, ConfIT, Entire CLI, Git AI, Google Antigravity, Google Mainframe Assessment Tool, OpenCode, OpenSpec, PageIndex, Pencil, Pi, Qwen 3 TTS, SGLang, ty, Warp.

## Caution Ring

**OpenClaw** - Hyper-personal AI assistant requiring extensive permission access, concentrating security risks through "permission-hungry" architecture patterns.
