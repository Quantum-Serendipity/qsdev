# Ccusage: A CLI tool for analyzing Claude Code usage from local JSONL files

**Submitter:** kristianp
**Repository:** https://github.com/ryoppippi/ccusage
**Website:** https://ccusage.com

## Key Discussion Points

### Cost & Sustainability Concerns
A primary user noted consuming $600-800 monthly on the $100/month Max plan while barely pushing limits. This sparked debate about Anthropic's actual margins and whether pricing resembles the business model challenges that affected Cursor.

### Model Quality & Reliability
Users praised Claude's instruction-following capabilities and consistency as key advantages. One developer described a workflow combining Opus for planning and implementation with O3 for review validation.

### Competitive Landscape
Discussion covered alternatives like Grok4, O3, and Gemini CLI. While competitors may benchmark higher on pure coding tasks, Claude's agentic design and discounted Max plan pricing provided distinct advantages.

### Usage Limit Changes
A TechCrunch article regarding tightened usage limits generated speculation about sustainability pressures.

### Security Considerations
Commenters raised concerns about running arbitrary tools with system access, with suggestions to use sandboxed environments like Docker containers for autonomous operations.
