# HN Discussion: The Overflow Offline Project (Oct 2022)

## Size and Compression

- SO ZIM shrank from 134G (2019-02) to 75G (2022-05) -- likely improved compression rather than removed data
- Raw SE data dumps: 486 GB data files with a 1.4 TB log file when loaded into SQL Server
- SQL Server instance lacked foreign key relationships or indexes on commonly queried columns

## Alternatives and Custom Builds

- Individual SE sites (non-SO) were already available as separate ZIMs before this partnership
- Arch Wiki also available as ZIM
- A user built "Kiwings" -- custom macOS app because Kiwix desktop had issues
- DevDocs (devdocs.io) noted as alternative for language/framework docs with offline IndexedDB support

## No Tag-Filtered Builds Mentioned

No commenters discussed building tag-filtered or subset versions of SO. The discussion focused on the full dataset.
