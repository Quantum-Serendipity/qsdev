# Show HN: Clawy, a companion device to track your Claude Code sessions

**Submitter:** mrcv
**Tool URL:** https://clawy.lol/
**GitHub Repository:** https://github.com/marcvermeeren/clawy
**Posted:** ~36 days ago (February 2026)

## Features & Functionality
- Clawy animates when Claude executes tools
- A celebratory jump indicates task completion
- Physical shake triggers a dazed animation
- When Claude requires permission, the device displays commands and context on-screen like "video game quest scrolling text"
- Users can approve or deny actions via physical buttons

## Technical Specifications
- **Hardware:** M5StickC Plus 2 (~$20)
- **Setup:** 2-minute browser-based flashing at clawy.lol/flash
- **No Arduino IDE required**
- **Communication:** Uses Claude Code's native hook system over local WiFi
- **Privacy:** "nothing leaves your network"

## Creator's Motivation
The developer initially built this as a personal productivity tool to monitor sessions while multitasking around their home, seeking an alternative to terminal interfaces.
