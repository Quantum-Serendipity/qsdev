# Cybersecurity ROI Calculation Methodology

## Key Frameworks

**FAIR (Factor Analysis of Information Risk)**
The framework breaks cyber risk into two components: loss event frequency (probability) and loss magnitude (impact). Organizations calculate expected annual loss for specific scenarios to compare risk reduction approaches.

**Gordon-Loeb Model**
Organizations can optimize security spending by investing up to 37% of the expected loss in protecting an asset. This provides a theoretical benchmark for evaluating investment efficiency.

## Core Formulas

**Annualized Loss Exposure (ALE)**
Formula: ALE = SLE x ARO

Where:
- **Single Loss Expectancy (SLE)**: Cost of one incident
- **Annualized Rate of Occurrence (ARO)**: How many times the incident occurs yearly

*Example*: Ransomware with $500,000 SLE and 20% occurrence rate (0.2 ARO) = $100,000 ALE

**Return on Security Investment (ROSI)**
Formula: ROSI = (Risk Exposure x Risk Mitigation % - Solution Cost) / Solution Cost

*Example*: $2M risk exposure reduced 70% by $150K training program = 833% ROSI

## Practical Application Steps

1. Identify critical assets and realistic threats
2. Estimate financial losses (direct costs, indirect costs, reputational damage)
3. Map security controls to risk reduction percentages
4. Calculate payback period: Initial Investment / Annual Risk Reduction

The methodology emphasizes translating technical metrics into business language for executive decision-making.
